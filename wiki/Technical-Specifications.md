# 🛠️ TECHNICAL SPECIFICATIONS

The terminal is a masterpiece of NERV engineering, built to survive the Third Impact while providing real-time data on Angel activity.

## 🏗️ Core Technology
- **Engine**: Vanilla HTML5, CSS3, and JavaScript (No external frameworks for maximum performance).
- **Styling**: NERV-themed CSS with glassmorphism, scanlines, and parallax effects.
- **Backend API**: Integrates directly with GitHub API for live commit telemetry.
- **Bot Integration**: Full support for Telegram Mini Apps (TMA) through `telegram-web-app.js`.

## 📡 Key Systems
### 1. Tactical 3D Globe
A lightweight CSS-based 3D engine that renders a rotating sphere with dynamic markers for Angel tracking. It uses `preserve-3d` and keyframe animations to simulate a globe without the overhead of Three.js.

### 2. A-T Field Defense (Mini-game)
A canvas-based real-time defense simulator. Intercept Angel Cores to maintain A-T Field integrity.

### 3. Bio-activity Heatmap
A Git-style activity grid that monitors the pilot's "physiological stability" over the last 52 weeks.
